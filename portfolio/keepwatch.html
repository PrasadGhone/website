Keepwatch: A safety and security system for elderlies staying alone using IoT and wearables

Keepwatch

The project was a class assignment for the Soft prototyping course. The problem was to design a system to solve problems faced by elderlies staying alone. After brainstorming, team finalised on solving safety and security issues faced by elderlies. A concept safety and security system was created using IoT and wearables.

Brainstorming

The first phase of the project was brainstorming on problems an elderly face staying alone.

Categorization

The problems from the brainstorming session were then categorized based on various parameters.

Problem Scenario

To emphathise with the problem scenario, we carried out experience prototyping. We enacted the scenario to get more insights by emphathy. Yes, I'm the old person in the video.

Concept Video

The concept video explains the solution space of the product and the usgae by the elderly in emergencies.

Product Explanation

This video explains all the features of the solution. We explored various 3d rendering softwares to create these mockups used in video.
