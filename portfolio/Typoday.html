Typopot- An Interactive Installation

Typopot is an interactive installation which was showcased in Typoday 2015, held at IIT Bombay. It is web based installation. It is an interactive game which is analogous to slot machine. Participant has to get a winning condition of all the letters of the type used in the identity of the event. When a trigger in a form of button is pressed, the notification is sent to web server, and the web application runs. Based on a logic to generate random numbers, the cards are shuffled. The user has to get the winning condition to win the game.
